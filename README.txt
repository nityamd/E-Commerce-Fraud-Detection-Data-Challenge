Instructions for viewing/running my data challenge submission in three
easy commands.

1. Install environment.

conda env create -f environment.yml

**warning:** this will attempt to make a conda environment named data_challenge.

2. Activate the new environment.

conda activate data_challenge

3. Load notebook from this directory.

jupyter notebook Case_Study_report.ipynb
